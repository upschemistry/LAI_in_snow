# LAI_in_snow
Repository for light-absorbing impurities in snow. Includes LAHM and ISSW data in separate folders. 

The LAHM filters include UCB and UPS samples, and are organized as follows:

"UCB UCB UPS" => UCB samples were analyzed on a UCB LAHM & analyzed by UPS code ("LAHM_analysis.ipynb")
"UCB UPS UPS" => UCB samples were analyzed on a UPS LAHM & analyzed by UPS code
"UPS UPS UPS" => UPS samples were analyzed on a UPS LAHM & analyzed by UPS code
